Import from JSON Import a JSON file that was previously exported from this application. Import from 5th Edition SRD Import a statblock from the Official D&D 5th Edition Systems Reference Document (SRD) from Wizards of the Coast. Import from Open5e Import a statblock from the Open5e API, which provides creatures from the following sourcebooks using the Open Game License:
  • Official 5th Edition SRD from Wizards of the Coast
  • Tome of Beasts from Kobold Press
  • Creature Codex from Kobold Press
Note: All text from Open5e will be imported as-is, and will not include any Markdown Emphasis, Name Expressions, or Math Expressions. Export to JSON Export your statblock to a JSON file that can be re-imported back at a later time. Export to HTML Export your statblock as a standalone HTML document, based on Valloric's statblock5e template.

This HTML document works best in Chrome. Export to Markdown Export your statblock to a special Markdown format for use in the following homebrew content tools:
  • Homebrewery
  • GM Binder

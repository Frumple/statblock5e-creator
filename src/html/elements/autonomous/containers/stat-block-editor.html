Import from Open5e Import your statblock from the Open5e API, which provides creatures from the following sourcebooks using the Open Game License:
  • Official 5th Edition SRD from Wizards of the Coast
  • Tome of Beasts from Kobold Press
  • Creature Codex from Kobold Press
Note: Imported text from Open5e will not include any Markdown Emphasis, Name Expressions, or Math Expressions. Import from JSON Import your statblock from a JSON file that was previously exported from statblock5e-creator. Export to JSON Export your statblock to a special JSON format that can be re-imported back into statblock5e-creator at a later time. Export to HTML Export your statblock as a standalone HTML document, based on Valloric's statblock5e template.

This HTML document works best in Chrome. Export to Markdown Export your statblock to a special Markdown format for use in the following homebrew content tools:
  • Homebrewery
  • GM Binder

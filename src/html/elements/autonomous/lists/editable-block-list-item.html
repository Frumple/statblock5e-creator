This text area supports basic Markdown formatting:

Emphasis (italic): *single asterisks* or _single underscores_
Strong Emphasis (bold): **double asterisks** or __double underlines__
Combined Emphasis: __*asterisks* nested in underlines__ and **vice _versa_**

This text area supports name expressions:

  • All instances of {name} are replaced with the creature's short name when displayed.
    If the short name is blank, the creature's full name is used instead.
  • All instances of {fullname} are replaced with the creature's full name when displayed.

If the creature is not a proper noun, the word "the" will precede the creature's short name/full name.

* For more details, see the help tooltips for "Short Name" and "Proper Noun" in the first section.

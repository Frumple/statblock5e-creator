Getting Started with Statblock5e Creator

Start with the default Commoner template, or use the Import menu to import a creature from the 5th Edition SRD or Open5e.

Click any section to start editing. To save your changes, click the [done] button to the right. Pressing the Enter key on most fields will also save the section.

Some fields will automatically update other parts of the statblock when changed. For example, changing ability scores will affect hit points, saving throws, skills, passive perception, and math expressions.

Use the Export menu to export your statblock to the following formats:

  • JSON, which can be re-imported back at a later time.
  • HTML, based on the original statblock5e template.
  • Markdown, for use in other homebrew content tools.

Advanced Syntax for Special Traits, Actions, Reactions, and Legendary Actions

Name Expressions

  • [NAME] - Shows the creature's full name, or short name if defined.

    If the creature's name is NOT a proper noun, the word "the" will precede the name.

    For example, given these conditions:
    • Full Name = "Adult Red Dragon",
    • Short Name = "dragon",
    • Creature is NOT a proper noun
    Then [NAME] = "the dragon".

Math Expressions

  • MOD[INT - 2] - Calculates the intelligence modifier minus 2.
  • ATK[STR] - Calculates the attack roll modifier for a strength-based attack.
  • DMG[2d6 + DEX] - Calculates the damage roll for a dexterity-based attack with 2d6 dice.
  • SDC[CHA] - Calculates the spell save DC with charisma as the spellcasting ability modifier.

Markdown Emphasis

Italic: *single asterisks* or _single underscores_
Bold: **double asterisks** or __double underscores__
Combined: __*asterisks* in underscores__ or **vice _versa_**

Skills

The creature's skill bonus is calculated using the following formula:

Skill Bonus = Ability Modifier [ + Proficiency Bonus if proficient in skill ]

If the creature has expertise in a skill, it adds double its proficiency bonus instead:

Skill Bonus = Ability Modifier + (Proficiency Bonus * 2)

For more details, see "Skills" on page 8 of the D&D 5th Edition Monster Manual.

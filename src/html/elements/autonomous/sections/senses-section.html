Senses

passive Perception 10

A creature's passive perception is calculated using the following formula:

Passive Perception = 10 + WIS Modifier [ + Proficiency Bonus if proficient in Perception ]

If you specify an override value for Perception in the Skill Proficiencies section, the passive perception is instead:

Passive Perception = 10 + Perception Override

For more details, see Passive Checks on Open5e, or on page 175 of the D&D 5th Edition Player's Handbook.

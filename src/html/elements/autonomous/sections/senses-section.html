passive Perception 10

A creature's Passive Perception is calculated using the following formula:

Passive Perception = 10 + WIS Modifier + Proficiency Bonus (if proficient in Perception)

Note: If you specify an override value for the Perception skill, the Passive Perception is 10 + the override instead.


* See "Passive Checks" on page 175 of the DnD 5e Player's Handbook for full details.
